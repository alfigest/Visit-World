Applications made with Flutter are world tourist destinations that feature:

- The destination
- Distance of tourist locations from the user's place
- Festifal tour
- Travel description
- The closest hotel from tourist attractions
- Cafe place
- The restaurant
- Application Mode (Dark, White)
